# Sistema de Recomendación MovieLens (Filtrado Colaborativo)

Este proyecto es la entrega para la **Práctica 4** de la asignatura *Gestión de Información en la Web* (Máster en Ingeniería Informática, UGR).

## 🎯 Objetivo
Desarrollar un sistema de recomendación Usuario-Usuario utilizando el dataset MovieLens 100k, permitiendo a nuevos usuarios recibir recomendaciones personalizadas basadas en similitudes con la base de datos histórica.

## 🛠️ Stack Tecnológico
- **Backend:** Python 3.10+, FastAPI, NumPy, Pandas (Cálculo matricial optimizado).
- **Frontend:** React + TypeScript + Tailwind CSS (Interfaz moderna e interactiva).
- **Algoritmo:** User-User Collaborative Filtering.
- **Similitud:** Coeficiente de Correlación de Pearson / Similitud de Coseno.
